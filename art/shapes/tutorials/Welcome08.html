Realtime Interaction

Ecstasy Motion provides some new tools on the bottom of the
Torque World Editor Tools Palette (pictured at left) for live
interactions between you and your actors.

The Grab Tool lets you pick up an actor. The Impulse
Tool lets you knock one down, with an arbitrary amount
of force specified at the top of the screen. The Move
Tool we'll talk about in a moment. The Explosion Tool
creates an outward force on all the actors within range.
Sorry about the render bug that happens with the grab tool - it doesn't show in recorded animations.
